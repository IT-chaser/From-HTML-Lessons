HTML дастурлаш тили хакида.

1989 йил  World wide web  яратилган ва айнан шу йили web хужжатлар асосий тили бўлган тил HTML (Hyper Text Markup Language - Гипер Матнларни Белгилаш Тили) яратилди.
HTML тилининг муаллифи – Оксфорд университетининг битирувчиси, Европанинг CERN (Conseil Europeen pour la Reserche Nucleaire) лабораториясида дастурий таъминотлар бўйича мутахассис Тим  Бернерс - Ли  (Tim Berners - Lee) дир.
HTML тили гиперматн технологиясининг асосини ташкил этади. Гиперматн хужжат гиперссылкаларни ўз ичига олади.
Матнли гиперссылкалар  одатда бошқа ранг билан ёки тагига чизиш билан ажратилади, лекин бу шарт эмас. Гиперссылкалар ёрдамида фойдаланувчи керакли хужжатнинг физик жойлашувидан қатъи назар  унга тез ўтиб кетиши мумкин.
Бу эса ер шарининг турли чеккаларидаги информацион қидиришни осонлаштиради.
Гиперматн технологияси Интернет глобал тармоғидаги хужжатларни ягона WWW тизимига бирлаштиради.
Хужжатларни белгилаш тили - Тэг деб номланувчи махсус кўрсатмалар тўплами бўлиб, у хужжатларда структурани тузиш ва структуранинг турли элементлари орасидаги боғланишни ифодалаш учун  хизмат қилади. Тилнинг тэглари  ёки бошқача қилиб айтганда бошқарувчи дескрпиторлари  бундай типдаги документларда қандайдир миқдорда кодланади, хужжатнинг асосий мазмунидан ажратилади.
Энг биринчи тизимларда бундай командаларни белгилаш учун  “<” ва “>” символлари ишлатилган. Ҳозирги кунда бу усул тэгларни белгилашнинг стандарт усули ҳисобланади.
Замонавий информацион тизимларда хужжатларни гиперматн сифатида тақдим этиш, информацияни ночизиқли кўриш имконини беради. Бундай тизимларда маълумотлар узилмас оқим сифатида эмас, аксинча бир-бири билан боғланган компонентлар кўринишида ҳосил бўлади, улар орасидаги  ўзаро ўтишлар эса гиперссқлкалар кўринишида бўлади.
Умуман олганда HTML тили дастурлаш тили эмас, у хужжат тузилиши, унинг стили, бошқа хужжатлар билан боғлиқлигини тасвирлаш воситасидир.WEB хужжатларни кўриш учун броузер деб номланувчи махсус дастурдан фойдаланилади.
HTML тилининг яратилиши 1989 йилда CERN лабораториясида маълумот алмашинувини енгиллаштирди, лекин бу билан бошқа шаҳарларда ишловчи лаборатория аъзолари билан маълумот алмашинуви таъминланмади. Бундай алмашинувни таъминлаш учун қандайдир стандарт асосида маълумот алмашинувини  таъминлайдиган, марказлашмаган (тарқалган) информацион тизим лозим эди. Бундай стандарт сифатида Тим бернерс томонидан 1990 йилда яратилган HTTP (Hyper Text Transfer Protocol) протоколи олинди.
HTML умумий стандартдаги белгилаш тили - SGML (Standard Generalized Markup Language ) нинг осонлаштирилган тури хисоланади, у 80 йилларда ISO томонидан тасдиқланган. Бу тил бошқа белгилаш тилларини яратиш учун мўлжалланган ва у рухсат этилган тэглар тўпламини, уларнинг хоссаларини, хужжатнинг структурасини аниқлайди.

SGML га қараганда содда ва қулай бўлган HTML тили хужжат элементларига безак бериш имконини беради ва тэгларнинг чегараланган кўрсатмалар тўпламига эга бўлади. HTML кўрсатмалари биринчи навбатда хужжат мазмунини тингловчи компьютер экранига чиқариш жараёнини бошқаришдир.
