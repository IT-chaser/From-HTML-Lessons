График файллар тушунчаса ва уларнинг турлари
Аввал озгина бўлса хам комаьютерда ишлаган бўлсангиз, унда турли график файллар мавжуд эканлигини билсангиз керак. Лекин шуни айтиб утиш жоизки, график файлларнинг турлари имкоятди кўп. Интернет билан ишлаш учун биз фақат уларнинг икки туридан, яъни JPEG ва GIF  турлари билан ишлаймиз Келинг, улар билан яқинроқ танишамиз демак, икки график файл тури ўзининг кучли томони билан бирга заиф томонларига хам эга.  GIF качик ўлчамга эга бўлиб, ўз таркибида 256 рангни саклай олади. Бу график файл туридан фақат график маълумотни сакдаш учун фойдаланишади, сабаби унинг жуда качик ўлчамга эга бўлишададир. JPEG каттароқ ўлчамга эга бўлади ва ўз таркибида 16.7 миллионгача рангни сақлай олади. Бу график файл туридан расмларни сақлашда фойдаланиш маъқул.

Front Page дастури атайин график файллар сақлаш учун Image номли каталогни олдиндан ўзи яратиб қўйган. Одатда, веб-сайт таркибига кирувчи барча график файллар айнан шу каталогда сакланади.
